Two numbers can be multiplied in any order (the commutative property of multiplication):
4 ( 5 )   =   5 ( 4 )  ;   16 · 8   =   8 · 16  ;   12 · 9   =   9 · 12
We can use the same properties to multiply a positive and a negative number.
Let us multiply 5 and – 6
5 ( – 6 )   =   ( – 6 ) + ( – 6 ) + ( – 6 ) + ( – 6 ) + ( – 6 )
  =   – 30
The same result can be obtained by placing “–” sign before the product:
5 ( – 6 )   =   – ( 5 · 6 )   =   – 30
With whole numbers,  ( 4 ) ( 5 )   =    5 ( 4 )    =   20
Similarly,  ( – 4 ) ( 5 )  =  5 ( – 4 )
 = ( – 4 ) + ( – 4 ) + ( – 4 ) + ( – 4 ) + ( – 4 )  = – 20
The same result can be obtained by placing the “–” sign before the product
( – 4 ) 5   =   – ( 4 · 5 )
  =   – 20
Note that the additive inverse (the opposite) of an integer a can be treated as – 1 times a;   that is,
– a   =   ( – 1 ) · a
The product of a positive and a negative number is the negative of the product of their absolute values.
This can be stated as:   “The product of two numbers with unlike signs is negative.”
That is,   ( + ) ( – )   =   ( – )   and   ( – ) ( + )   =   ( – )
For example,
1223122312235 ( – 4 )   =   – ( 5 · 4 )   and   ( – 7 ) ( 4 )   =   – ( 7 · 4 )

The product of two numbers with like signs (that is, both positive or both negative) is positive, and is equal to the product of their absolute values.
This can be stated as:   “The product of two numbers with like signs is positive”.
That is,   ( + ) ( + )   =   ( + )     and     ( – ) ( – )   =   ( + )

To estimate products, each number is rounded to its largest place value. The largest place value is always on the left of the number. This is called front-end rounding.
To front-end round – 382, we round to the nearest hundred, – 400
To estimate the product of – 3,982 and 43, front-end round each factor.
Then multiply.
– 3,982 ≈ – 4,000
4 zeros
× 0043 ≈ × 0040
– 160,000
4 zeros
The estimate of the product is – 160,000. The actual product is – 171,226.

Procedure for estimating a product of: 472 and 8,593.

472 ≈ 500

Therefore, estimated product of 472 and 8,592 is 4,500,000.

Commutative Property of Multiplication
For any two numbers a and b;
a · b = b · a that is,
the order of numbers in multiplication can be reversed.
Examples:
●
5 ( – 3 ) = ( – 3 ) 5(-53321.)

●
( – 7 ) ( – 8 ) = ( – 8 ) ( – 7 )
The product on each side is 56.
Associative Property of Multiplication
For any three numbers a, b, and c;
( a · b ) · c = a · ( b · c )
the grouping of the factors can be changed.
Examples:  [ 4 · ( – 5 ) ] · ( – 6 ) = 4 · [ ( – 5 ) · ( – 6 ) ]

Identity Property for Multiplication
For any number a;
a · 1 = a;  1 · a = a;
The product of 1 and the number is the same number.
The number 1 is called the multiplicative identity.
Examples:
●
( – 2 ) · 1 = – 2
Examples:
●
1 · ( – 3 ) = – 3
Examples:
●

Zero Factor Property
For any number a;
a · 0 = 0;  0 · a = 0
The product of number and 0 is always 0.
Examples:
●
0 · ( – 3 ) = 0
●
  0 · ( + 6 ) = 0
Distributive Property of Multiplication over Addition or Subtraction
If a, b, and c are any three numbers, then
a · ( b + c ) = a · b + a · c;
( a + b ) · c = a · c + b · c
that is, multiplication distributes over addition.
a · ( b – c ) = a · b – a · c;
( a – b ) · c = a · c – b · c
that is, multiplication distributes over subtraction.
Examples:   ( – 3 ) · ( – 2 + 5 ) = ( – 3 ) · ( – 2 ) + ( – 3 ) · 5
