A numerical expression is a combination of numbers, operations (+, –, ×, ÷), and exponents.
Now we shall learn to simplify numerical expressions which may involve any combination of these operations on Integers.
To do this, it is necessary to have some laws or rules of order in which the various operations should be performed, so that people, calculators, and computers get the same results.
In the absence of such rules, it is possible to interpret even simple numerical expressions like 4·5+7 in two different ways :
4 · 5 + 7
or
4 · 5 + 7
=
20 + 7
First multiply
=
4 · 12
First add
=
27
then add
=
48
then multiply
We get two different results from the same numerical expression.
In order to avoid such a situation, universally accepted rules for the order of operations are used.

RULES FOR SIMPLIFYING NUMERICAL EXPRESSIONS INVOLVING GROUPING SYMBOLS
Rule 1:
Simplify any expression(s) within grouping symbols first.
If grouping symbols are imbedded then start with the innermost grouping.
Rule 2:
If an expression involves different operations, perform these operations in the order given below :
◆
Simplify the exponents first, from left to right.
◆
Then simplify multiplications and divisions, from left to right.
◆
Finally, simplify additions and subtractions from left to right.
Let us simplify the following expression:
25–6÷( – 3 )+8·4
=
25–( – 2 )+32
=
25+2+32
=
59
Most calculators are programmed to follow the rules for order of operations.
Using a scientific calculator, the procedure is simple.
Just enter all the symbols in the given expression in the order you see them from left to right.
