The symbol for equality “=” is read “is equal to”.
Inequality symbols are symbols that are used to express a relationship between two numbers.
The symbol = is read “is equal to”.
The symbol ≠ is read “is not equal to”.
The symbol < is read “is less than”.
The symbol > is read “is greater than”.
The symbol ≤ is read “is less than or equal to”.
The symbol ≥ is read “is greater than or equal to”.
To round a whole number means to give an approximate value of the number to a given rounding place.
The number 457,249 rounded to the tens place is 457,250.
457,249 ≈ 457,250.

A Number Line is a line on which numbers are represented by points.
The whole numbers can be represented by points on a number line as follows:
Draw a line and mark a point on it. Label this point with the number 0.
Beginning with 0, make marks to the right of 0 at equal intervals, label them by the numbers 1, 2, 3, 4, …
The arrow means that the numbers continue forever.
